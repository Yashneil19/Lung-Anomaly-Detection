# Lung-Anomaly-Detection
Adventitious breath sounds are superimposed on normal respiratory sounds and serve as key di
agnostic indicators in pulmonary medicine. Manual auscultation is the clinical gold standard for
their detection, but it is subjective, time-consuming, and dependent on clinician experience. Auto
mated detection systems that analyse digital lung sound recordings offer the prospect of objective,
reproducible screening tools that can be deployed at scale.

This project  goal was to develop frame-level classifiers for four adventitious sound categories — Wheeze, Rhonchi, Stridor,
and Crackle — using acoustic feature engineering and machine learning. The work proceeded
iteratively: early experiments on wheeze established the foundational analysis framework, which was
then adapted and extended to the remaining three sound types before all classifiers were consolidated
into a single multiclass detection system
